Slide the sensor, not the world.

Three objects sit at different distances in front of a fixed lens. The sensor can slide along the optical axis. As you move it, watch which object snaps into focus and which ones dissolve — and see directly in the viewfinder what the camera records at each position.

What you're seeing

Each object casts a cone of rays into the lens. The lens redirects each cone so it re-converges at that object's specific image distance — every object has its own "correct" sensor position.

The sensor can only sit at one position at a time, so only one depth plane can be perfectly sharp. Everything else is a circle of confusion — a cone of rays sliced before (or after) it has finished converging.

Real cameras solve this by physically moving the lens (not the sensor) forward and back — but mathematically it's identical to what you're doing here. The focus ring is the user interface for this slider.
